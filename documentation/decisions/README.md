# Decisions

Architecture Decision Records (ADRs) documenting significant technical choices.

## Naming convention

`{YYYY-MM-DD}-{decision-slug}.md` — date-prefixed for chronological ordering.

## Frontmatter template

```yaml
---
title: "ADR: Decision Title"
date: YYYY-MM-DD
author: agent | username
status: draft | active | superseded | deprecated
related_issues: []
related_mrs: []
---
```

See `TEMPLATE.md` in this folder for the full ADR structure.
