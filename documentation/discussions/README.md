# Discussions

Discussion protocols, meeting notes, and conversation summaries.

## Naming convention

`{YYYY-MM-DD}-{topic}.md` — date-prefixed for chronological ordering.

## Frontmatter template

```yaml
---
title: "Discussion Title"
date: YYYY-MM-DD
author: agent | username
status: draft | active | superseded | deprecated
related_issues: []
related_mrs: []
---
```
