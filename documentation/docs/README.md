# Docs

Feature documentation, how-to guides, and technical references.

## Naming convention

`{feature-name}.md` — no date prefix, use descriptive kebab-case names.

## Frontmatter template

```yaml
---
title: "Document Title"
date: YYYY-MM-DD
author: agent | username
status: draft | active | superseded | deprecated
related_issues: []
related_mrs: []
---
```
