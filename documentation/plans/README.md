# Plans

Project plans, roadmaps, and implementation strategies.

## Naming convention

`{plan-name}.md` — no date prefix, use descriptive kebab-case names.

## Frontmatter template

```yaml
---
title: "Plan Title"
date: YYYY-MM-DD
author: agent | username
status: draft | active | superseded | deprecated
related_issues: []
related_mrs: []
---
```
